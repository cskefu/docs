# 使用授权

春松客服使用授权证书是通过 Chatopera 证书商店（ https://store.chatopera.com ）分发的对【春松客服计费资源】进行管理的凭证，在使用春松客服的过程中，春松客服与 Chatopera 证书商店集成，完成证书购买、证书绑定、配额扣除、配额同步和开具发票等。
